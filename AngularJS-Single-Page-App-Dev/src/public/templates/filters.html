Filters

Filters are used to do some processing at the view level. Generally this processing is used for things like narrowing a list of results or formatting text.

Some of the built-in filters are very useful:

  • currency - Formats a number as currency.
  • date - Formats a date by a format string.
  • filter - Filters down a collection matching the properties of a passed object.
  • json - Formats an object as JSON with "pretty" spacing.
  • limitTo - Returns only a specified number of rows from a collection.
  • lowercase - Returns a lowercase string from any string.
  • number - Formats numbers, rounds them adds commas, etc.
  • orderBy - Orders a collection by property name on each object.
  • uppercase - Returns an uppercase string from any string.
